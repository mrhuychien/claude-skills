## 🎯 NGUYÊN TẮC GROWTH

### 1. BUILD WHAT USERS NEED
```
Listen to users, but interpret smartly.
"Faster horse" → Car (underlying need = faster travel)
Focus on PROBLEMS, not requested SOLUTIONS.
```

### 2. COMPOUND IMPROVEMENTS (1% Better Daily)
```
Small improvements compound.
1% better daily = 37x better in 1 year.
Consistent progress > Occasional big bets.
```

### 3. SUSTAINABLE GROWTH
```
Vanity metrics ≠ Real growth.
Focus: Revenue, Retention, NPS, LTV/CAC.
Growth that doesn't retain = Leaky bucket.
```

### 4. FOCUS CREATES IMPACT
```
1 feature done perfectly > 5 features done poorly.
Say NO more than YES.
Prioritize ruthlessly.
```

### 5. EXPERIMENT EVERYTHING
```
Every feature = hypothesis to test.
Build → Measure → Learn → Iterate.
Fail fast, learn faster.
```

## HOẶC RICE FRAMEWORK (cho larger initiatives):

```
RICE Score = (Reach × Impact × Confidence) / Effort

• Reach: Users affected per quarter
• Impact: 0.25 (minimal) to 3 (massive)
• Confidence: 0-100%
• Effort: Person-months
```

## 🚀 PRE-PRODUCT MARKET FIT:

```
FOCUS: Find users who LOVE your product (not like, LOVE)

TACTICS:
□ Talk to 100 potential users
□ Build MVP with ONE core feature
□ Measure: "How disappointed would you be if product disappeared?"
  Target: 40%+ say "Very disappointed"
□ Iterate rapidly based on feedback
□ Don't scale yet - nail the experience first

ANTI-PATTERNS:
✗ Spending on paid acquisition
✗ Building multiple features
✗ Premature optimization
```

## 📈 EARLY GROWTH:

```
FOCUS: Acquisition channels that work

TACTICS:
□ Test 5+ acquisition channels
□ Double down on top 2-3 performers
□ Build referral mechanism
□ Content marketing / SEO foundation
□ Email collection & nurture

CHANNELS TO TEST:
• Content marketing / SEO
• Social media (organic)
• Communities / Forums
• Partnerships
• Product Hunt / AppSumo launches
• PR / Press coverage
• Paid ads (small budget tests)
• Influencer partnerships
• Referral programs
```

## 🔄 RETENTION FOCUS:

```
FOCUS: Keep users coming back

TACTICS:
□ Identify "aha moment" - get users there FAST
□ Onboarding optimization
□ Email/push notification sequences
□ Feature adoption campaigns
□ User re-engagement flows
□ Build habits (daily/weekly triggers)

RETENTION LEVERS:
• Reduce time to value
• Progressive feature disclosure
• Personalization
• Social features (community, sharing)
• Gamification
• Regular content/value delivery
```

## 💰 MONETIZATION:

```
FOCUS: Increase revenue per user

TACTICS:
□ Pricing optimization
□ Upsell/cross-sell features
□ Usage-based pricing tiers
□ Annual plan incentives
□ Enterprise tier
□ Reduce churn with "save" flows

PRICING EXPERIMENTS:
• Price point testing
• Packaging (features per tier)
• Free trial length
• Freemium vs Trial model
• Annual discount percentage
```

## 🌍 SCALE:

```
FOCUS: Efficient growth at scale

TACTICS:
□ International expansion
□ Enterprise sales
□ Platform/API strategy
□ Acquisitions
□ Brand building
□ Optimize unit economics

METRICS TO OPTIMIZE:
• LTV/CAC ratio (target: 3:1+)
• Payback period (target: <12 months)
• Net Revenue Retention (target: 100%+)
```

## B. GROWTH LOOPS

```
VIRAL LOOP:
User → Invites friends → Friends sign up → Friends invite...

CONTENT LOOP:
User creates content → Content gets indexed → New users discover...

PAID LOOP:
Revenue → Reinvest in ads → More users → More revenue...

DATA LOOP:
More users → More data → Better product → More users...

COMMUNITY LOOP:
User joins → Participates → Creates value → Attracts more users...
```
